A- O calor latente da fusão desses sólidos é equivalente a 15000 Joules.

B- Os calores específicos dos estados liquido e solido do material são de 133,2J/Kg e 1000J/Kg.

Como sabemos, são fornecidos 10 KJ/min, com isso:

Q1 = mct

10000 = 500*c*(10-(-5))

10000 = 500*c*15

10000 = 7500c

c = 10000/7500

c = 1,33J/g ou 133,2J/Kg

No intervalo de 1,5 min, são 15000 Joules

Q2 = mL

15000 = 500*L

L = 15000/500

L = 30J/g ou 30000J/Kg

O terceiro intervalo , que é de 1,5 min, são 15000 Joules

Q1 = mct

15000 = 500*c*(40-10)

15000 = 500*c*30

15000 = 15000 c

c = 15000/15000

c = 1,0J/g ou 1000 J/Kg
